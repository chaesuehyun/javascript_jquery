01 자바스크립트_문서출력하기

자바스크립트에서는 다양한 방식으로 데이터를 출력하는 방법이 있습니다

  1. documnet객체에 write 메소드를 사용 - document.write('출력할 내용');
  2. window객체를 활용하여 메세지 띄우기 - window.alert('메세지 내용');
  3. 브라우저 개발자 모드의 console 모드 - console.log('출력할 내용');
  4. document객체의 innerHTML메소드로 내용 출력하기 - document.getElementById('아이디명').innerHTML ='출력할 내용';
