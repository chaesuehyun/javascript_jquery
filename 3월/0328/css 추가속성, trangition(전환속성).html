css 추가속성, trangition(전환속성)

transition(전환속성)은 사용자가 선택한 html요소에 마우스를 올리거나 클릭을 하였을 때 첫장면에서 마지막 장면으로 자연스렇게 넘어가는 효과를 구현하고자 할 때 사용한다.

  • 모바일 토글메뉴 클릭시 부드럽게 나오는 내비게이션
  • pc화면에서 메인메뉴를 마우스 오버시 부드럽게 펼쳐지는 서브메뉴목록
  • 메인이미지 좌, 우 버튼 클릭시 부드럽게 옆으로 애니메이션되는 기능
  • 메인이미지 컨트롤 버튼 클릭시 부드럽게 사라지거나 나오는 기능
  • 상품사진에 마우스 오버시 설명내용이 아래에서 위로 부드럽게 올라오는 기능
  • 버튼에 마우스 오버시 부드럽게 변하는 배경색상
