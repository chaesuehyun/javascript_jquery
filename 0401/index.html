반응형 (response web)웹 이란? 하나의 html문서를 제작하여pc, tablet, mobile화면에서 최적화된 콘텐츠를 보여주기 위한 기술이다.

  1. 가변단위 -em, rem, %, vh, vw등등
  2. 뷰포트 속성 - mate name='viewport'
  3. 미디어 쿼리 - 해상도별 css적용하기

반응형 웹 - 디비컷,'https://mediaqueri.es'

내비게이션 영역

2.뷰포트 속성(viewport)

사용자의 디바이스 화면크기 (pc, tablet, mobile)에 맞추어 웹문서를 100%로 보여 줄 수 있는 속성으로서 pc보다 태블릿이나 모바일 사용자를 위해서 필요하다.

웹문서의 상단 head 에 meta name="viewport" content ="width =divice-width, initial-scale=1'내용을 삽입한다

  • name="viewport" : pc, tabletm mobile 디바이스 화면에 맞게 설정함
  • width=divice-width : 콘텐츠 내용을 pc tablet mobile 화면 가로너비와 일치시킴
  • initial-scale=1 : 디바이스 화며네 1:1비율로 콘테츠를 보이도록 함. 숫자 '2'를 입력하면 화면두배크기로 커짐

3 미디어 쿼리 속성

브라우저의 가로너비 (해상도)에 따라 또는 사용자의 디스플레이 가로 해상도에 따라 css서식을 적용하기 위한 기술로서 반응형 웹페이지 제작시 필수 속성인다

해상도 참고 사이트 : https://screensize.es/phone, http://iosres.com

미디어 쿼리 작성시 각각의 디바이스별 해상도에 대한 이해가 필요한데, 모바일 영역을 0~ 767ㅔㅌ, 태블릿 영역을 768~ 1024px, pc 영역은 1025px 이상~ 해상도 영역을 주로 설정한다. 각 디바이스멸 해상도 기준을 정하는 것을 'break Point(중단점)'이라고 한다

https://gs.statcounter.com

          미디어 쿼리 작성법
          @media screen and (디바이스별 가로 해상도) {
            조건에 맞는 해상도일 때 적용할 css서식;
          }
