실습 1 _채팅창 구현

채팅창 입력양식을 삽입하여 사용자가 글내용 입력 후 '전송'버튼 클릭시 글내용이 대화상자에 출력되게 하기

  • 채팅창 화면을 header, main , footer 영역으로 구성
  • header영역은 브라우저 버튼, 사용자가 프로필영역, 소개글 작성
  • main영역은 상대방, 사용자가 입력한 글내용이 말풍선으로 출력되는 곳
  • footer영역은 사용자가 입력양식에 글을 입력하고 버튼 클릭시 변수에 담아 내용을 출력하도록 함.
  • 이때, 사용자가 글 내용을 입력하지 않고, 버튼을 클릭하면 disabled 속성으로 클릭안되게 하기
  • 속성 메서드 : val 클래스 조작 : addClass 내용추가 : append
