자바스크립트_수학객체

수학객체는 다른 내장 객체와는 다르게 new 키워드를 사용하기 않는 정적인 객체인다.

예) let today = new Date(); : 날짜 객체 생성

예) Math.변수명

객체(object)는 메서드(동작), 속성(명사)로 나누어진다

메서드 : 기능적인 동작을 말하며 write();가 대표적인 메서드이다.

속성 : 서식등을 말하며 style.color='서식종류'가 대표적인 속성이다.

1. 수학객체의 속성들

    e : 상수

    PI : 원주율

    SQRT2

    SQRT1_2

    LN2

    LN10

    LOG2E

    LOG10E

수학객체의 메서드 종류

  • max(숫자1, 숫자2.......) : 숫자들 중에서 최대값을 반환
  • min(숫자1, 숫자2.......) : 숫자들 중에서 최소값을 반환
  • round(숫자) : 소수점 첫째 자리를 반올림
  • ceil(숫자) : 소수점 자리를 무조건 올림
  • floor(숫자) : 소수점 자리를 무조건 절삭
  • abs(숫자) : 수의 절대값을 반환
  • random(숫자) : 0과 1 사이의 난수를 표시함
