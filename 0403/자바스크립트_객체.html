자바스크립트_객체(object)

객체란
1. 이름 (key)과 값(value)로 구성된 속성을 담아놓은 컨테이너 이다.
2. 객체에 변수를 선언하면 속성(property)을 사용할 수 있다. 예)documnet.getElementById('아이디명');
3. 객체에 함수를 선언하면 메서드(method)를 사용할 수 있다. 예)documnet.write('내용출력');

1. 객체의 종류

객체는 내장객체와 사용자 정의 객체로 분류된다. 내장 객체는 자바스크립트에 미리 정해놓은 객체이다.

  • 문서객체(document)
  • 윈도우, 브라우저 객체(window)
  • 수학(Math)객체
  • 날짜 (Date)
  • 배열 (Array)
  • 문자열 (String)

2. 사용자 정의 객체 - 내장겍체를 제외한 사용자가 원하는 내용을 토대로 직접 객체를 선언하여 작성하는 방법
